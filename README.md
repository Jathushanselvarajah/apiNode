# simple_api

This is a simple API client project that demonstrates how to set up a Node.js application with MySQL integration. The project includes a package.json file for managing dependencies, a seed script for populating initial data, and a server.js file that implements CRUD routes for managing clients.

## Getting Started
1. Clone the repository
2. Install dependencies
   ```
   npm install
   ```
3. Run the migration script to set up the database
    ```
    npm run migrate
    ```
4. Run the seed script to populate initial data
    ```
    npm run seed
    ```
5. Start the server
    ```
    npm run start
    ```
